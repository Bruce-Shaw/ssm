# ssm

开发工具为idea，数据库为mysql8.0.12，使用Spring(4.3.5.RELEASE)+SpringMVC+MyBatis(3.4.1)开发模式

## 配置

### 总体目录结构
* `src` 源代码目录
* `target` 输出目录，所有的输出都存放在这个目录下
* `pom.xml` 用于maven的配置文件

## 开发

### 目录结构
* `src/main` 工程源代码目录
* `src/main/java` 工程java源代码目录
* `src/main/resource` 工程的资源目录
* `src/test` 单元测试目录
